---

# 프로젝트 소개

이 프로젝트는 대학 네트워크 수업의 **TCP/IP 전송계층(Transport Layer)** 을 하나의 Markdown 교재로 정리한 자료입니다.

교수님의 PDF를 기준으로 내용을 재구성하였으며,

단순한 요약이 아니라

**처음 배우는 학생도 이해할 수 있는 교재**를 목표로 작성되었습니다.

GitHub와 Obsidian에서 그대로 사용할 수 있도록 Markdown 형식으로 작성하였습니다.
