# 핵심 요약

- TCP는 데이터를 보내고 ACK를 받으며 Window를 앞으로 이동시킨다.
- ACK 번호는 다음에 받을 Byte 번호를 의미한다.
- SND.UNA는 ACK를 받으면 앞으로 이동한다.
- SND.NXT는 전송할 때 증가한다.
- Window 크기만큼만 데이터를 전송할 수 있다.
- 이 과정을 반복하면서 대용량 데이터를 효율적으로 전송한다.
